Images

this page covers common image formats. You may also be looking for links and uxeful extras

JPGs

JPG is good for

  • handling large color palettes
  • keeping low file size
  • can handle images with many gradients

However, JPG does not support transparrent pixels

GIFs

GIFs are great for simple animations, but should not be used for images.
They do not have much color palette support.

PNGs

PNG is great for diagrams, icons, and can support transparrent pixels
A PNG and a JPG of the same perceived quality would have a wildly different file size.

SVGs

SVG uses vector based graphics (like adobe illustrator), and can scale up/down willy nilly.

Despite being mostly better than PNGs, they require much workaround for consistent display
